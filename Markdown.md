# MARKDOWN
Markdown é uma linguagem de marcação de texto leve e simples,
criada para formatar documentos usando caracteres fáceis de ler,
que depois são convertidos para HTML ou outros formatos. 

## Titulos e Subtitulos
Para destacar o titulo do arquivo vai de 1 # ate 6 #;
Semelhante ao h1 de html.

## ÊNFASE
**Negrito** ou __Negrito__ -> 2 asteristicos ou 2 underline.
*Italico* ou _Italico_ -> 1 asteristico ou 1 underline.

## Listas
__Listas não ordenadas (ul):__
* Item1
     * Item1.1(Sublista)
* Item2             
__Listas ordenada(ol):__ 
1. Item1
     * Item1.1(Sublista)
2. Item2
